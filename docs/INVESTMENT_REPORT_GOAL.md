# 투자형 미국장 보고서 목표

## 최종 목표

매일 아침 카카오톡으로 짧은 링크를 받고, 웹 보고서 하나만 열어도 그날의 투자 판단에 필요한 핵심을 확인할 수 있게 만든다.

보고서는 단순 뉴스 요약이 아니라 아래 질문에 답해야 한다.

1. 오늘 시장은 공격적으로 볼 수 있는가, 방어적으로 봐야 하는가?
2. 어떤 섹터를 우선 보고, 어떤 섹터를 피해야 하는가?
3. 내 보유/관심종목은 시장 흐름보다 강한가, 약한가?
4. 후보 종목의 매수 타점, 손절 타점, 매수 근거, 손절 근거는 무엇인가?
5. 어제 후보는 오늘 기준으로 맞았는가, 틀렸는가, 아직 관찰 중인가?
6. 실적, 경제지표, 공시 같은 이벤트 리스크가 가까운가?

## 구현 범위

- 오늘의 결론
  - 시장 모드
  - 우선 볼 섹터
  - 조심할 섹터
  - 핵심 테마
  - 행동 원칙

- 오늘 매매 가능 점수
  - 0~100점으로 당일 시장 환경 우호도 표시
  - 선별 매수 가능 / 관찰 우위 / 관망 우위 / 방어 우선 등급 표시
  - 점수는 매수 추천이 아니라 시장 환경 판단용으로 사용

- 전문 투자자 체크
  - 공격적 관찰 / 성장주 선별 / 선별 관찰 / 방어 우선 중 하나로 당일 태도 표시
  - 신규 진입 강도 표시
  - 오늘 확인할 순서 표시
  - 무효화 조건을 명시해 추격 매수를 막는다

- 오늘의 경고
  - 매수하지 말아야 할 이유를 별도 섹션으로 표시
  - 방어주 주도, 성장주 약세, VIX/금리/달러 부담을 경고로 표시

- 핵심 리스크와 촉매
  - VIX, 금리, 달러, 유가의 긍정/부정 신호 구분
  - 주요 뉴스의 긍정/부정 톤을 투자 판단 언어로 정리

- 뉴스 품질 개선
  - 뉴스마다 중요도 A/B/C 등급 표시
  - 뉴스 방향과 가격 반응이 일치하는지 표시
  - 같은 의미의 뉴스가 반복되지 않게 제한

- 섹터 로테이션 판정
  - 주도 섹터와 약한 섹터를 함께 표시
  - 상승/하락 섹터 개수로 시장 폭 확인
  - 성장주 주도, 방어주 주도, 성장주 약세 같은 해석을 붙인다

- 후보 종목 점수화
  - 0~100점 점수
  - 점수 등급
  - 점수 이유
  - 매수 타점
  - 손절 타점
  - 매수 근거
  - 손절 근거

- 전일 후보 추적
  - 전일 관심 후보의 현재 성과
  - 전일 비선호 후보의 현재 성과
  - 매수 타점 도달 여부
  - 손절 기준 이탈 여부

- 이벤트 캘린더
  - FRED API 키가 있으면 고용보고서, CPI, PCE, FOMC, GDP 발표 일정을 자동 표시
  - 키가 없으면 설정 필요 상태와 체크해야 할 이벤트 목록 표시

- 실적 발표 캘린더
  - Alpha Vantage API 키가 있으면 WATCHLIST_SYMBOLS의 다음 실적 발표일 표시
  - 실적 전 신규 진입 주의 여부 표시
  - 키가 없으면 발급/설정 필요 상태 표시

- 관심종목 SEC 공시 알림
  - WATCHLIST_SYMBOLS에 넣은 종목의 최근 8-K, 10-Q, 10-K, 6-K, 20-F 공시 확인
  - 최근 주요 공시가 없으면 없다고 명시

- 보유/관심종목 영향 분석
  - WATCHLIST_SYMBOLS에 넣은 종목을 섹터 흐름과 연결
  - 종목 움직임과 섹터 움직임이 같은지 다른지 해석
  - 포트폴리오 섹터 쏠림과 상대강도 표시

- 웹 보고서 개선
  - 카카오톡 원문이 아니라 보고서 섹션으로 표시
  - 결론, 이벤트, 점수, 추적 결과를 상단에서 빠르게 확인

## 원칙

- 너무 많은 정보를 넣기보다 투자 판단 순서가 보이게 만든다.
- 모든 후보에는 타점과 무효화 기준이 있어야 한다.
- 추천이 맞았는지 다음 날 자동으로 검증한다.
- 카카오톡은 짧게, 웹 보고서는 자세히 보여준다.
- 개인 맞춤 투자자문이 아니라 규칙 기반 참고자료임을 유지한다.

## 이후 확장 후보

- FINRA margin debt, VIX term structure 같은 과열/레버리지 지표
- 과거 후보 성과 누적 대시보드
- 섹터별 ETF 자금 흐름
- 보유 수량과 비중을 입력받는 실제 포트폴리오 위험 관리
