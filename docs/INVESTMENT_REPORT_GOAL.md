# 투자형 미국장 보고서 목표

## 최종 목표

매일 아침 카카오톡으로 짧은 링크를 받고, 웹 보고서 하나만 열어도 그날의 투자 판단에 필요한 핵심을 확인할 수 있게 만든다.

보고서는 단순 뉴스 요약이 아니라 아래 질문에 바로 답해야 한다.

1. 오늘 시장은 공격적으로 봐야 하는가, 방어적으로 봐야 하는가?
2. 어떤 섹터를 우선 보고, 어떤 섹터를 피해야 하는가?
3. 관심 후보와 비선호 후보는 무엇이며, 왜 그렇게 판단했는가?
4. 매수 타점, 손절 타점, 매수 근거, 손절 근거는 무엇인가?
5. 어제의 후보는 오늘 기준으로 맞았는가, 틀렸는가, 아직 관찰 중인가?

## 1차 구현 범위

- 오늘의 결론 박스
  - 시장 모드
  - 볼 섹터
  - 피할 섹터
  - 오늘 행동 원칙
- 후보 종목 점수화
  - 0~100점 점수
  - 점수 등급
  - 점수를 만든 핵심 이유
- 전일 후보 추적
  - 전날 관심 후보의 오늘 등락
  - 전날 비선호 후보의 오늘 등락
  - 매수 타점 도달 여부
  - 손절 타점 이탈 여부
- 웹 보고서 개선
  - 카카오 원문이 아니라 보고서 섹션으로 표시
  - 결론, 점수, 추적 결과를 상단에서 빠르게 확인

## 이후 확장 후보

- 보유 종목 입력 후 보유 종목 영향 분석
- CPI, FOMC, 고용지표, 실적 발표 캘린더
- SEC 공시 기반 관심 종목 이벤트 감지
- FINRA margin debt, VIX term structure 같은 과열/레버리지 지표
- 과거 후보 성과 누적 대시보드

## 원칙

- 너무 많은 정보를 넣기보다 투자 판단 순서가 보이게 만든다.
- 모든 후보에는 타점과 무효화 기준이 있어야 한다.
- 추천이 맞았는지 다음 날 자동으로 검증한다.
- 카카오톡은 짧게, 웹 보고서는 자세히 보여준다.
- 개인 맞춤 투자자문이 아니라 규칙 기반 참고자료임을 유지한다.
